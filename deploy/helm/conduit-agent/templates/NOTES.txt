Send a test trace
-----------------
From any pod in the cluster:

  curl -X POST http://{{ include "conduit-agent.fullname" . }}:{{ .Values.service.ports.http }}/v1/traces \
    -H 'Content-Type: application/json' \
    -d '{"resourceSpans":[{"resource":{"attributes":[{"key":"service.name","value":{"stringValue":"smoketest"}}]},"scopeSpans":[{"spans":[{"traceId":"01020304050607080102030405060708","spanId":"0102030405060708","name":"smoke","startTimeUnixNano":1,"endTimeUnixNano":2}]}]}]}'

Check it landed:

  kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "conduit-agent.name" . }} --tail=20

Probe pod liveness:

  kubectl exec -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "conduit-agent.name" . }} -- /usr/bin/conduit --version || true
  # Health check via kubectl port-forward:
  kubectl port-forward -n {{ .Release.Namespace }} ds/{{ include "conduit-agent.fullname" . }} 13133:13133 &
  curl -s http://127.0.0.1:13133
